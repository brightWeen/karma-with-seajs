### 说明
seajs使用karma做单元测试 

### 安装
执行`npm install` 

### 运行
执行测试并且启动文件监听,在修改seajs模块或者测试脚本时会自动重新执行测试

    karma start
    
仅执行测试
    
    karma start karma.conf.js --singleRun
